# Redfinix

Netflix-inspired real estate frontend built with React and Tailwind CSS.

## Stack

- React
- React DOM
- React Router DOM
- Tailwind CSS
- Lucide React
- Axios
- PropTypes

## Installed Packages

- `react`
- `react-dom`
- `react-router-dom`
- `tailwindcss`
- `@tailwindcss/vite`
- `lucide-react`
- `axios`
- `prop-types`
- `vite`
- `@vitejs/plugin-react`
- `eslint`
- `@eslint/js`
- `eslint-plugin-react-hooks`
- `eslint-plugin-react-refresh`
- `globals`

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Create a production build:

```bash
npm run build
```

## Current Notes

- Trending slider components use `PropTypes`, so `prop-types` is required.
- UI is being built section by section with a Netflix-style visual direction.
